# Monitoramento e Alertas StreamPay

Este documento descreve o fluxo de monitoramento automatizado das integrações externas do backend StreamPay AI, incluindo Moralis, Etherscan, Infura, ElizaOS, KYC, SMTP e logs de contrato.

## Visão Geral
  - ElizaOS (healthz)
  - Moralis (saldo)
  - Etherscan (transações)
  - Infura (gas)
  - KYC (status)
  - SMTP (teste de envio)
  - Logs de contrato (eventos críticos)
