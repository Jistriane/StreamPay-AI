# StreamPay Monitoring and Alerts

This document describes the automated monitoring flow for the StreamPay AI backend integrations, including Moralis, Etherscan, Infura, ElizaOS, KYC, SMTP, and contract logs.

## Overview
  - ElizaOS (healthz)
  - Moralis (balance)
  - Etherscan (transactions)
  - Infura (gas)
  - KYC (status)
  - SMTP (send test)
  - Contract logs (critical events)
